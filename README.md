# EmeraldsVector

A dynamic list using fast memory management.

# Installation

- Install the library

`em list`
`em install`
`em build lib`

## Usage

- Include in a file

```c
#include "EmeraldsVector.h"

int main(void) {
  /* Create a new vector object */
  int **v = vector_new(1);

  vector_add(v, 2);
  vector_add(v, 3);

  printf("%d\n", v[1]);
  printf("%d\n", v[0]);
}
```
